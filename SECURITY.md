# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in these NVIDIA patches, please report it by emailing:

**scott@elyanlabs.ai**

Please include:
- Description of the vulnerability
- Affected patch files
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will respond within 48 hours and work with you to address the issue.

## Security Best Practices

When using these patches:
- Verify patch integrity before applying
- Test in a safe environment first
- Keep your system updated
- Review patch contents carefully
- Report suspicious behavior

Thank you for helping keep this project secure!
